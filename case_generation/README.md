# 医学病例自动生成工具

## 项目简介

本项目是一个医学病例自动生成工具，能够根据用户输入的一段描述症状、疾病或治疗的文本，自动生成结构化的医学病例。

### 功能特点

- 支持从简短的症状描述生成完整的结构化病例
- 自动识别文本中的医学实体（疾病、症状、药物、治疗方法等）
- 生成标准化的病例格式，包括主诉、现病史、既往史、体格检查等部分
- 提供交互式命令行界面，使用简单方便

## 安装说明

### 环境要求

- Python 3.7+
- pip 19.0+

### 安装依赖

1. 进入项目目录：

```bash
cd case_generation
```

2. 安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

### 运行主脚本

```bash
python main.py
```

### 输入示例

程序运行后，会提示用户输入文本。用户可以输入一段描述症状、疾病或治疗的文本，例如：

```
患者因头痛、发热3天入院，伴有恶心、呕吐症状，既往有高血压病史，需要进行血液检查。
```

### 输出结果

系统会自动处理输入文本，识别医学实体，并生成结构化的病例，包括基本信息、主诉、现病史、既往史、体格检查、辅助检查、诊断和治疗方案等部分。

### 退出程序

输入 `exit` 退出程序。

## 配置说明

项目的配置信息存储在 `config/config.json` 文件中，包括：

- `medical_entity_types`：医学实体类型映射
- `case_template`：病例模板结构
- `keyword_lists`：关键词列表，用于基于关键词的实体识别

用户可以根据需要修改这些配置信息。

## 技术实现

### 文本处理模块

使用 NLTK 库进行文本预处理、分词和关键短语提取。

### 医学实体识别模块

- 优先使用专业的医学 NER 模型（en_medical_ner）
- 如果医学模型不可用，使用通用 NER 模型（en_core_web_sm）
- 提供基于关键词的实体识别作为备选方案

### 病例生成模块

基于识别到的医学实体和输入文本，按照标准病例格式生成结构化病例。

## 注意事项

- 本工具仅作为辅助工具，生成的病例需要医生根据实际情况进行审核和修改。
- 医学实体识别的准确性取决于使用的模型和关键词列表的覆盖范围。
- 对于复杂的病例描述，可能需要手动调整生成的病例内容。
