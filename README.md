# 项目介绍
包含以下项目：
- 医疗术语自动补全

# 环境准备

## 硬件要求
- **CPU**：至少4核CPU
- **内存**：至少8GB RAM（加载BERT模型需要）
- **存储**：至少10GB可用空间（用于存储模型文件）
- **GPU**：可选，用于加速训练和推理

## 软件要求
- **Python**：3.12.9
- **pip**：25.0

目前均属于demo状态，后续会持续更新。
